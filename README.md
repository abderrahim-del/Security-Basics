# Security-Basics

Dieses Projekt enthält grundlegende Informationen über IT-Sicherheit.

## Phishing

Phishing ist eine Methode, bei der Angreifer versuchen,
Passwörter oder persönliche Daten zu stehlen.

Oft geschieht das durch gefälschte E-Mails oder Webseiten.

## Malware

Malware ist schädliche Software.

Beispiele:
- Viren
- Trojaner
- Spyware

Malware kann Daten stehlen oder Computer beschädigen.

## Firewall

Eine Firewall schützt ein Netzwerk vor unerlaubtem Zugriff.

Sie kontrolliert den Datenverkehr zwischen einem Computer
und dem Internet.

## Sichere Passwörter

Ein gutes Passwort sollte:

- lang sein
- Buchstaben, Zahlen und Symbole enthalten
- nicht leicht zu erraten sein

Beispiel:
T7!kL9@pQ
